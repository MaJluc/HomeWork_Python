# Задание 1

## 🔍 Найдите ошибку в коде:

``` python
from flask import Flask

app = Flask(__name__)

@app.route('')
def home():
    return 'Hello, World!'

if __name__ == '__main__':
    app.run()
```

------------------------------------------------------------------------

## ❌ Ошибка находится в строке:

``` python
@app.route('')
```

------------------------------------------------------------------------

## ✅ Правильный вариант

``` python
@app.route('/')
```

------------------------------------------------------------------------

## ✔ Исправленный код:

``` python
from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return 'Hello, World!'

if __name__ == '__main__':
    app.run()
```
